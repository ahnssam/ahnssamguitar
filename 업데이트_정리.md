# 안쌤의 기타나무숲 — 업데이트 정리

> TWA 앱 / 설문조사용. 이번에 개선·추가된 내용을 사용자 입장에서 정리한 문구입니다.

## 한눈에 보기
이번 업데이트에서는 **랭킹**, **반주(코드 메이커)**, **메인 화면**이 크게 좋아졌습니다.
특히 반주 도구에 **악보 인쇄/PDF 저장**, **송폼(곡 구성) 표시**, **한 마디 4코드 입력**, **피아노 사운드 개선**이 새로 들어갔습니다.

---

## 1. 연습도구 메인 화면
- 메뉴를 **한 줄에 3개씩** 깔끔하게 정렬하고, 메뉴 박스를 화면 가운데로 정돈했습니다.
- '키별 화음', '피아노' 등 일부 메뉴 아이콘을 좌측 메뉴 아이콘과 **통일**했습니다.

## 2. 도구 링크 공유 개선
- 카카오톡 등으로 특정 도구 링크를 공유하면 **그 도구가 바로 열립니다.**
  (이전에는 공유하면 연습도구 메인만 떴어요.)

## 3. 랭킹
- **연속(스트릭) 랭킹도 분야별로** 볼 수 있습니다. (전체 · 음 맞추기 · 두 음 비교 · 코드 맞추기)
- 랭킹에서 **아이디를 누르면**, 분야별로 보고 있을 때도 그 사람이 **어떤 난이도로 점수를 올렸는지** 보입니다.
- **'고수' 랭킹**이 새로 생겼습니다. 각 분야의 **최고 난이도 기록만** 모아 보여줍니다.
  (코드 맞추기는 '지옥', 그 외 분야는 '어려움' 기준)
- 랭킹 화면 위쪽에서 **종합 / 고수**를 인터넷 탭처럼 눌러 전환할 수 있고, 점수·연속 둘 다 지원합니다.
- 다크 모드에서 탭 글씨가 어둡게 묻히던 부분을 **잘 보이게** 고쳤습니다.

## 4. 반주 (코드 메이커)
- **악보 인쇄 / PDF 저장** 기능이 생겼습니다.
  - 인쇄 버튼을 누르면 **제목·아티스트를 입력**하고 **미리보기**까지 보고 인쇄할 수 있어요.
  - 화면의 코드 진행(코드·도돌이·볼타·구성)이 **그대로** 출력됩니다.
  - 맨 위 가운데 제목, 그 아래 오른쪽에 아티스트가 표시됩니다.
  - 크롬에서는 인쇄 대신 **'PDF로 저장'**도 됩니다.
- **송폼(곡 구성) 표시** 기능 추가. 마디 번호를 누르면 '전주 · 후렴 · 브릿지'처럼 **자유롭게 입력**해 마디 위에 표시할 수 있습니다.
- **한 마디에 코드 2개 / 4개** 전환이 가능해졌습니다. (기본 2개)
  - 4개로 하면 한 박마다 코드를 넣을 수 있고, 입력·악보·인쇄·**재생**까지 모두 반영됩니다.
  - 4코드 모드에서는 코드를 넣은 박을 **깔끔하게 한 번씩** 연주합니다.
- **피아노 사운드 개선** — 직접 녹음한 피아노 음원을 사용할 수 있도록 시스템을 마련했습니다. (베이스·드럼처럼 실제 녹음 음 사용)
- 마디 표시도 다듬었습니다. **마디 세로 크기를 일정하게** 맞추고, 도돌이 표시가 마디 번호와 겹쳐 누르기 불편하던 부분을 **고쳤습니다.**
