루프스테이션 개발 데모

루프스테이션

사용법
사용법
  1. 왼쪽 준비 버튼으로 마이크를 켭니다. (다시 누르면 감지 중지)
  2. RECORD 또는 R 키를 누르면 신호 대기 상태가 되고, 소리가 들어오면 자동으로 녹음이 시작됩니다.
  3. 녹음 중 같은 키를 다시 누르거나 아무 키나 누르면 정지 + 즉시 반복 재생됩니다. 첫 녹음이 루프 길이의 기준이 돼요.
  4. 루프가 도는 동안 다시 녹음하면 그 위에 겹쳐서(오버더빙) 녹음됩니다. 한 바퀴 후 자동 정지.
  5. 오토 더빙 켜면 녹음 직후 자동으로 다음 오버더빙 대기로, 연속 더빙 켜면 정지할 때까지 계속 겹쳐 녹음합니다.
  6. 파형 양옆 핸들을 드래그하면 루프 시작·끝점을 자를 수 있어요(폰 녹음 편집처럼).
  7. 스페이스바=PLAY(재생/일시정지), REVERSE 역재생, UNDO 되돌리기.
입력 레벨감지 임계값 -40 dB
— dB
RECORD 를 누르면 파형이 표시됩니다
볼륨 100%
속도 100%
