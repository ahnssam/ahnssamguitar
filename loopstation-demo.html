루프스테이션 개발 데모

루프스테이션

사용법
  1. 아래 시작 버튼으로 마이크 권한을 허용해주세요.
  2. 녹음할 트랙의 큰 동그란 버튼을 누르면 신호 대기 상태가 됩니다.
  3. 마이크에 일정 크기 이상의 소리가 들어오면 자동으로 녹음이 시작됩니다.
  4. 녹음 중에 아무 키나 누르면 즉시 정지 + 루프 재생이 시작됩니다.
  5. 다른 트랙도 같은 방식으로 쌓아 올리면 됩니다. 첫 트랙이 루프 길이의 기준이 되고, 이후 트랙은 그 배수(1×/2×)로 자동 정렬돼요.
입력 레벨
— dB
-40 dB
시작 버튼을 눌러 마이크를 켜주세요.
